# Plants Leaf Detection System 🌿

An AI-powered Plant Leaf Detection System that identifies plant leaf diseases using Machine Learning and Image Processing techniques.

## 🚀 Project Overview

The Plants Leaf Detection System helps users detect and classify diseases in plant leaves by analyzing uploaded leaf images. The system uses Machine Learning / Deep Learning algorithms to provide accurate predictions and assist farmers and researchers in monitoring plant health.

## ✨ Features

- Detect plant leaf diseases
- Image-based classification
- AI-powered prediction system
- User-friendly interface
- Fast and accurate results
- Supports multiple plant categories

## 🛠 Technologies Used

- Deep Learning
